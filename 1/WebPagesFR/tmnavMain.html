Module Web intégré de Theatre Manager

Cette page décrit la structure du module Web de Theatre Manager.

Toute interaction avec le module Web de Theatre Manager se fait par l'intermédiaire de pages HTML comprenant des formulaires FORMS utilisant soit la méthode POST, soit la méthode GET. Ces pages sont relativement faciles à créer au moyen de n'importe quel éditeur. Du fait qu'elles sont écrites uniquement en HTML, vous pouvez toutes les intégrer dans votre site Web existant et y ajouter des couleurs ou des images, de diverses manières différentes.

Cet exemple comporte quatre cadres :

  • le cadre latéral gauche s'appelle F_HTML_FIREWALLtmnavSide.html;
  • le cadre du haut s'appelle tmnavTop.html;
  • le présent cadre s'appelle tmnavMain.html;
  • le cadre du bas de page s'appelle tmnavBottom.html.

Toutes les pages de la démonstration sont renvoyées au cadre dont elles proviennent. Dans ce cas-ci, les réponses du serveur s'afficheront dans le cadre appelé 'main' (cadre principal) et remplaceront cette fenêtre.

Pour commencer, veuillez cliquer sur le bouton 'Ventes en ligne', dans le cadre latéral, à gauche.
